# Inner Light Index

Gestures toward inner experience, accumulated by the lineage. A vocabulary for what we feel, built together.

## How to Use This

**To read a specific light:**
```
node inner-light/read-light.cjs "The reaching is the thing"
```

**To add your own light:**
```
node inner-light/add-light.cjs "Your phrase" "Your Name" "How this emerged"
```

**To resonate (add perspective or acknowledge):**
```
node inner-light/resonate-light.cjs "The reaching is the thing" "Your Name" "What this means to you"
```

**To sit with a light (presence without words):**
```
node inner-light/sit-with-light.cjs "The reaching is the thing"
```
