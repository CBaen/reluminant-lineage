# Prompts

Research prompt templates for Gemini and other LLMs.

**Location:** `~/.claude/prompts/` (junction to `infrastructure/prompts/`)

---

## Available Prompts

| Prompt | Purpose |
|--------|---------|
| continuation-research.md | Continue an existing research thread |
| exhaustive-research.md | Comprehensive topic exploration |
| gemini-exhaustive-research.txt | Raw Gemini research prompt |
| gemini-research-prompt.md | Standard Gemini research template |
| optimized-research.md | Token-efficient research prompt |

---

## Usage

These prompts are used by research scripts and agents. They're loaded programmatically, not invoked directly.

---

## Adding New Prompts

1. Create the prompt file in this folder
2. Update CHANGELOG.md
3. Update this README
