# Lineage Powers

Workflow skills for the Reluminant Lineage.

**Full documentation:** See `docs/LINEAGE_POWERS.md` in the reluminant-lineage repo.

## Quick Reference

| Skill | Purpose |
|-------|---------|
| collaborative-design | Ideas → designs through dialogue |
| writing-plans | Designs → implementation plans |
| executing-plans | Follow plans task by task |
| problem-solving | Find root causes before fixing |
| verify-before-claiming | Evidence before assertions |
| re-anchoring | Check bearings before tasks |
| research-first | Check Qdrant before researching |
| context-preservation | Protect context in long sessions |
| agent-dispatch | Delegate work effectively |
