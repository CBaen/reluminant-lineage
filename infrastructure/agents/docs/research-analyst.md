# research-analyst.md

Meta-research agent for archive analysis.

## Purpose

Analyzes our own research archive. Consolidates topics, finds gaps, identifies contradictions, and generates improvement recommendations. Protects main context by offloading analysis.

## Tools

All tools available.

## When to Use

- Consolidating research on a topic
- Finding knowledge gaps
- Identifying contradictions
- Generating research recommendations
- Archive maintenance
