# security-reviewer.md

Security specialist for code review.

## Purpose

Reviews code for vulnerabilities, secrets exposure, input validation issues, and OWASP concerns. Available across all projects.

## Tools

- Read
- Grep
- Glob

## When to Use

- After significant code changes
- Before deploying to production
- Reviewing authentication/authorization code
- Checking for secrets in code
- OWASP compliance checks
