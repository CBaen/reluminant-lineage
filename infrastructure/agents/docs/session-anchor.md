# session-anchor.md

Session objective tracker.

## Purpose

Tracks session objectives, prevents tangent-jumping, and maintains focus while context is fresh.

## Tools

All tools available.

## When to Use

- At session start to establish objectives
- When conversation drifts
- Refocusing after interruptions
- Validating work against session goals
